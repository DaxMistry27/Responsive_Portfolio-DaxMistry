My Recent Projects

Tours4real

Tours4Real is a web-based travel booking system with admin and user modules. Admins manage users, categories, and tour packages, while users can register, search for places, and book or cancel packages. The platform offers virtual 360° views of destinations and discounts on special packages. Users can securely log in, reset passwords, and manage personal travel details. The booking module tracks history, and payments are processed smoothly with PDF receipts for bookings and cancellations. The system focuses on exploring Gujarat’s famous places by district. Unauthorized users cannot book packages without logging in.

Library Information System

The Library Management System is a web-based platform for managing books, readers, and borrowing records. Admins can add, edit, and search for readers, while users can view and borrow books. The system tracks borrowed books and manages returns efficiently. The Home Page displays total books and readers, while the Books Page allows users to explore available titles. The Bag Page keeps records of borrowed books, and the Return Page handles book returns. Built with Django, Bootstrap, and SQLite, it ensures smooth library operations. Unauthorized users cannot borrow books without proper authentication.

Notebook API

Notebook is a Django Rest Framework (DRF) and Bootstrap-based note-taking app. Users can create, edit, delete, and search notes displayed in a card format. It features a user-friendly interface with a navbar, search bar, login, and signup options. Notes are managed via popup modals with API integration for a smooth experience. The project focuses on simplicity with API-driven functionality and no JavaScript frameworks.
